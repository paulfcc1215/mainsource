ACTA DE LEVANTAMIENTO DE REQUERIMIENTOS

- MODELADO DE BASE DE DATOS QUE SOPORTE STORAGE DE LOS SIGUIENTES CONCEPTOS:
  - PERSONAS
  - TELEFONOS
  - CLASIFICACION DE TELEFONOS (LISTAS BLANCAS, LISTAS GRISES, LISTAS NEGRAS)
  
- PROCESO DE INSERCIÓN DE DATOS
  - CAPA DE ABSTRACCIÓN (WEB SERVICE) PARA INSERCIÓN DE DATOS
    Este proceso permitirá inserción desde cualquier fuente de datos, siempre y cuando el cliente de inserción consuma la capa de abstracción.
  - CONTROL DE INSERCIÓN DE DATA, CON LAS SIGUIENTES VALIDACIONES
    - DUPLICIDAD DE PERSONAS POR CEDULA
    - DUPLICIDAD DE TELEFONOS POR CEDULA Y TELEFONO

- INTERFAZ DE ADMINISTRACION
  - CARGA DE BASES AL REPOSITORIO (DEFINIR FORMA DE CARGA, PUEDE SER POR MAPEO O POR CABECERA DE ARCHIVO)
  - MODULO DE CONSULTA DE CLIENTES POR CEDULA

PRODUCTOS ENTREGABLES

 - MODELO FISICO DE BASE DE DATOS
 - SCRIPT DE CREACIÓN DE BASES DE DATOS
 - CODIGO FUENTE DE IMPLEMENTACION

REQUERIMIENTOS TÉCNICOS

- INFRAESTRUCTURA TECNOLOGICA
  - SERVIDOR FISICO O VIRTUAL, CARACTERISTICAS RECOMENDADAS
    - RAM 8 GB
    - CPU 4 CORES
    - DISCO 500 GB
    
- INFRAESTRUCTURA DE SERVICIOS
  - SISTEMA OPERATIVO CENTOS 7
  - SERVIDOR DE BDD POSTGRES 11
  - SERVIDOR WEB APACHE CON PHP 7

COMPROMISOS ADQUIRIDOS
  - DURANTE 2 SABADOS SE PROCEDERA CON UNA REVISIÓN DE AVANCE Y CONTINUIDAD DE DESARROLLO EN SITIO
  - "PCINFOSOL" SE COMPROMETE A DISTRIBUIR CODIGO FUENTE Y SCRIPT DE BASE DE DATOS A "ATERRIZAR GLOBAL"
  - "ATERRIZAR GLOBAL" SE COMPROMETE A PONER EN PRODUCCIÓN LA SOLUCIÓN INFIRMÁTICA EN SUS SERVIDORES
